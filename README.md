# Distributed File Storage.
Based on P2P network, You can store a file and distribute over the network simultaneously. 
Data is also encrypted over the network and hashed key.

## Debug Commands.

### To run.
```
    make run 
```
### To run the test.
```
    make test
```
### To build the application.
```
    make build
```
